Creating Your First Book

Joomla 1.5.x compatible version

This tutorial takes you through the steps of creating your first book and helps you understand how the component works. Please note that some of parameters are explained next to the input fields – just take your time!
Technical support does not cover issues relating to the incorrect operation of the component resulting from hacks, mods, patches and similar alterations to the Joomla! source code.

The basic steps for creating a book on your website are:

1. Installation.
2. Uploading Book Files.
3. Creating the Book.
4. Adding Pages to the Book.
5. Binding Book, Book List or Categories List to Menu Items

1. Installation

The standard installation and deinstallation of the component is covered in the official documentation available from http://docs.joomla.org

1.1. To install the component call up the Joomla! administration interface.
1.2. Go to the component installation tab (Extensions > Install/Uninstall).
1.3. Select the component file to be uploaded from your computer (com_flippingbook_personal_for_1.5.zip for example). Click Upload File & Install.
1.4. Make sure no errors are reported during installation.
If there are errors please check that Joomla folders are accessible for writing (Help > System Info > Directory Permissions) and PHP Safe Mode is turned off (Help > System Info > PHP Settings > Safe Mode: off) 1.5. Installation is complete. The component is available from the main menu of the administration panel (Components > FlippingBook).

2. Uploading Book Files

FlippingBook files are stored in /images/flippingbook folder.

2.1. Prepare the files containing the book pages. These can only be JPG images or SWF animations. Please use only Latin letters or numbers in the file names.
Do not use any international encodings.
No special symbols are allowed.
We recommend creating meaningful filenames – e.g. my-book_001.jpg, my-book_002.jpg... my-book_zoom_001.jpg, my-book_zoom_002.jpg...2.2. Go to the file file manager (Components > FlippingBook > Manage Files).
2.3. Create a new folder for your book (Create a new folder link). Open the created folder.
2.4. Select the file for uploading from your computer and click Upload.

3. Creating the Book

3.1. Go to the Book Manager tab (Components > FlippingBook > Manage Books).
3.2. Click New in the upper panel.
3.3. Enter the book name and description, if necessary.
3.4. Fill in other book parameters - their descriptions are given on the same page.
3.5. Click Save from the upper panel. A message will appear if the fields are not filled out properly.

4. Adding Pages to the Book

Manual Adding Pages

4.1.1. Go to the Page Manager (Components > FlippingBook > Manage Pages).
4.1.2. Click New in the upper panel.
4.1.3. Select the page file and the book to which the page is to be added.
4.1.4. Click Save in the upper panel.
4.1.5. Repeat the above steps for all the book pages.
4.1.6. Check the page numbers. If they are wrong use the Up and Down arrows to set them properly.
4.1.7. If necessary, specify the enlarged image file and the page link.

Batch Adding Pages

4.2.1. Prepare files for your book. Files should be named e.g. my-book_001.jpg, my-book_002.jpg... for pages and my-book_zoom_001.jpg, my-book_zoom_002.jpg... for zoom.
4.2.2. Go to Components > FlippingBook > Batch add Pages.
4.2.3. If you don't going use zoom feature - select your book, folder with uploaded files and mode > simple. Click Save.
4.2.4. If you want use zoom feature - set mode > advanced, Prefix For Pages Images > my-book_, Prefix For Zoomed Images > zoom_. Click Save.

5. Binding Book, Book List or Categories List to Menu Items

5.1. Go to the Menu Editor (Menu > target menu where you want the book reference to appear).
5.2. Click New in the upper panel.
5.3. Click FlippingBook. Select one of layouts. Specify parameters in a right column (Book Name for Book Layout or Category Name for Book List Layout). Click Save.

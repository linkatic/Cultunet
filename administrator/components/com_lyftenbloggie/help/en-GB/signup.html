LyftenBloggie: Signup Page

The following outlines the signup page for Tumbuh

Layout Positions

Title:

This is the page's title.

Introduction:

This is where you can highlight some of the features you API functions offer. It is recomended that you use this portion of the signup page. A blank page would just be ugly.

Terms and Conditions:

This is not needed, but it is a good way to inform your users of what conduct will get them banned from using your API functions. If this section is not fillout, the "I have read and agree with the terms and conditions" checkbox will not appear.

User Agrement & URL box:

This is the section where the user can enter their sites URL and agree to your terms and conditions
